Time-Series Analysis

Frequency Analysis and Variability Diagnostics

Development of Python workflows for analysing variability in time-series data, including frequency extraction, periodogram-based diagnostics, uncertainty-aware comparisons, and visual interpretation of complex signals.

This work involves signal-processing methods, batch analysis across multiple targets, and producing clear figures for both exploration and publication.

Scientific Pipeline Development

Spectroscopic Modelling and Automated Analysis Pipelines

Design of analysis pipelines for high-resolution spectroscopic datasets, including data ingestion, preprocessing, line-profile diagnostics, radial-velocity workflows, broadening analysis, and structured output for downstream modelling.

A major focus is building robust, reusable code that can handle many spectra efficiently while maintaining transparency and reproducibility.
